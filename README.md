# Belá Fiore — Catálogo Digital

## Sobre o projeto

O Belá Fiore é um site institucional desenvolvido para uma floricultura, com foco na apresentação de produtos de forma organizada, visual e acessível. O objetivo principal é oferecer uma vitrine digital moderna, facilitando a visualização do catálogo e a comunicação com clientes.

O projeto foi pensado para ser leve, responsivo e de fácil navegação, proporcionando uma experiência fluida tanto em dispositivos móveis quanto em desktop.

## Objetivos

- Apresentar o catálogo de produtos de forma clara e organizada

- Melhorar a presença digital da floricultura

- Facilitar o acesso às informações e produtos

- Oferecer uma interface simples e intuitiva para os clientes

## Funcionalidades

- Página inicial com destaque visual da marca

- Seção de catálogo de produtos

- Apresentação organizada dos itens da floricultura

- Layout responsivo para diferentes tamanhos de tela

- Estrutura simples e de fácil manutenção

## Tecnologias utilizadas

- HTML5

- CSS3

- JavaScript (quando aplicado)

- Hospedagem na plataforma Vercel

## Deploy

O projeto está hospedado na plataforma Vercel e pode ser acessado diretamente pelo link:

[Belá Fiore](https://belafiore.vercel.app/)


## Observações

Este projeto foi desenvolvido com foco em praticidade e performance, podendo ser expandido futuramente com novas funcionalidades como painel administrativo, integração com banco de dados ou sistema de pedidos.
